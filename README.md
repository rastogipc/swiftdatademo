A simple iOS application demonstrating basic CRUD operations using SwiftData (Apple’s modern persistence framework).
This project is intentionally minimal and focuses only on:
➕ Add Data
✏️ Update Data (Hardcoded)
❌ Delete Data
🚀 Features
Add new records using SwiftData
Update existing records with hardcoded values
Delete records from local database
Lightweight and beginner-friendly implementation
Built using SwiftUI + SwiftData
🛠️ Tech Stack
Language: Swift
UI Framework: SwiftUI
Persistence: SwiftData
Architecture: MVVM (Basic)
